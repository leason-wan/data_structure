# 数组

## Introduction

数组是在内存中分配一块连续的内存空间。  
一个数组的各个维度的下标与内存中的存储单元有着一一对应，这是必然的。这个对应关系称作映像函数。  

## 特性

- 连续的内存空间
- 计算各元素位置的时间相同，存取任意元素时间都是单位O(1)
- 删除中间一项时，后置元素会向前补位
